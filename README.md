# Expense Tracker

A simple Expense Tracker built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## Features

- Add new expenses
- Delete expenses
- Filter expenses by category
- Display all expenses
- Real-time statistics
  - Total Expenses
  - Total Amount
  - Highest Expense
  - Lowest Expense
  - Average Expense

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
